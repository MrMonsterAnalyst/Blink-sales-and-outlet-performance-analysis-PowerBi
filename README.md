# Blink-sales-and-outlet-performance-analysis-PowerBi
Domain: Retail Analytics / FMCG / E-Commerce
Tools: Power BI (DAX, Data Modeling, Power Query), Excel (Data Cleaning)

Project Overview

Developed an interactive retail performance dashboard in Power BI to analyze Blinkit’s outlet sales, product performance, item visibility, and customer ratings. The dashboard enables management to evaluate revenue drivers, outlet tier performance, and product category contribution using dynamic filters and KPI monitoring.

Key Contributions & Insights

Modeled and analyzed $1.20M total sales, 8.52K items, and $140.99 average sales value using structured data modeling and DAX measures.

Built dynamic KPI cards tracking Total Sales, Avg Sales, No. of Items, and Avg Rating (3.9).

Analyzed Outlet Establishment trends (2012–2022) to identify revenue growth patterns and peak-performing expansion years.

Evaluated sales contribution by outlet size (Medium, Small, High), identifying Medium-tier outlets as the highest revenue contributors (~42%).

Performed location tier analysis (Tier 1, Tier 2, Tier 3), showing Tier 3 locations generating the highest sales (~$472K).

Conducted category-wise performance analysis, identifying top revenue-driving item types (Fruits & Vegetables, Snack Foods, Household).

Compared Low Fat vs Regular product sales, enabling product mix evaluation.

Built interactive slicers for Outlet Location Type, Outlet Size, and Item Type for dynamic business exploration.

Created structured relationships and optimized DAX measures for accurate cross-filtered reporting.

Business Impact

Enabled data-driven identification of high-performing outlet tiers and product categories for expansion focus.

Provided clarity on product mix profitability trends, supporting assortment planning decisions.

Helped simulate decision-making for location-based revenue optimization.

Replaced static reports with an interactive BI solution, improving executive-level visibility into retail performance metrics.
